☠ NH Lab — PWA Deploy Guide

Turn nethunter-lab.html into an installable phone app in 5 minutes. Free. No coding.

FILES YOU HAVE

nethunter-lab.htmlMain app (PWA-ready)
manifest.jsonApp name, icon, colors
sw.jsService worker (offline support)
icon-192.pngHome screen icon
icon-512.pngSplash screen icon

OPTION A — GitHub Pages (Recommended)

01

Go to github.com → Sign up free → Click New Repository → name it nhlab → set Public → Create.

02

Click uploading an existing file → drag all 5 files into the box → click Commit changes.

03

Go to repo Settings → Pages → Source → Deploy from branch → main → / (root) → Save.

04

Wait ~60 seconds. Your app is live at:
https://YOUR-USERNAME.github.io/nhlab/nethunter-lab.html

05

On your Android phone, open that URL in Chrome → tap the 3-dot menu → "Add to Home screen" → Install. Done — it's an app icon on your phone!

💡 After installing, the app works 100% offline — no internet needed. The service worker caches everything on first load.

OPTION B — Termux Local Server (Instant, No Internet)

# Copy files to Termux storage
$ cp nethunter-lab.html ~/storage/downloads/nhlab/
$ cp manifest.json sw.js icon-192.png icon-512.png ~/storage/downloads/nhlab/

# Serve locally
$ cd ~/storage/downloads/nhlab
$ python3 -m http.server 8080

# Open Chrome on phone:
# http://127.0.0.1:8080/nethunter-lab.html
# Then: 3-dot menu → Add to Home screen

OPTION C — Netlify Drop (2 Minutes, No Account Needed)

01

Go to app.netlify.com/drop

02

Put all 5 files into a folder called nhlab → drag the entire folder into the Netlify drop zone.

03

Netlify gives you a random URL instantly — open it in Chrome on your phone → Add to Home screen.

INSTALLING ON iOS (Safari)

01

Open the URL in Safari (not Chrome — iOS PWA install only works in Safari).

02

Tap the Share button (box with arrow) → "Add to Home Screen" → Add.

⚡ The app icon will show the green skull ☠ icon on your home screen and launch fullscreen — looks and feels like a native app.
